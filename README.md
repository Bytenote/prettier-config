# @bytenote/prettier-config

Reusable Prettier configuration.

## Usage

**Install:**

```sh
# for NPM
npm i -D @bytenote/prettier-config
```

```sh
# for PNPM
pnpm i -D @bytenote/prettier-config
```

```sh
# for Bun
bun i -D @bytenote/prettier-config
```

**Add to package.json:**

```jsonc
{
    // ...
    "prettier": "@bytenote/prettier-config",
}
```
